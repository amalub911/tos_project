# 🚨 ToS Gotcha Highlighter

A machine learning-powered tool that detects and highlights **unfair or risky clauses** in Terms of Service agreements.

## 🔍 Features

* 🧠 BERT-based clause classification
* 🌐 Browser extension for real-time highlighting
* ⚡ Flask backend for inference
* 📊 Trained on labeled ToS dataset

## 🏗️ Tech Stack

* Python (Flask)
* Transformers (BERT)
* JavaScript (Chrome Extension)
* HTML/CSS

## ⚙️ Setup

### 1. Clone repo

git clone https://github.com/amalub911/tos_project.git

### 2. Install dependencies

pip install -r backend/requirements.txt

### 3. Download model

👉 [(https://drive.google.com/file/d/1PlRjVYKfKfpMYGjvbi5I6cHmBrol8j1G/view?usp=drive_link)]

Place it inside:
backend/tos_model/

### 4. Run backend

python backend/app.py
